# Cobblemon Server (Docker)

Fabric-based Cobblemon server using Docker.

## Requirements
- Docker + Docker Compose

## Start
in terminal:
docker compose up -d

## Stop
in terminal:
docker compose down

## Notes
- Minecraft: 1.21.1
- Loader: Fabric
- Modpack: cobblemon-fabric 1.7.1
- Java: 21
- Default port: 25565
- Default UDP Port: 24454
- Will not restart unless stopped
